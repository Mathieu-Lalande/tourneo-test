# Tourneo Test -- Ce fichier est généré par IA

## Informations Générales

Ce projet est un exemple d'application pour la gestion de tournois. Il inclut des fonctionnalités de création, d'inscription, de gestion des phases et de démarrage de tournois.

## Structure du projet

- `src/` : Code source principal
  - `tournament.js` : Logique métier des tournois
  - `tests/` : Tests unitaires
    - `tournament.creation.test.js`
    - `tournament.phases.test.js`
    - `tournament.registration.test.js`
    - `tournament.start.test.js`

## Lancement des tests

Pour exécuter les tests, utilisez la commande suivante :

```bash
npm test
```
